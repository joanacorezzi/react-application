# Book Finder

## Description
Book Finder is a simple React app that lets users search for books by title and view basic book details. It uses the Open Library API.

## Setup Instructions
1. Clone the repository
2. Install dependencies:
   - npm install
3. Start the dev server:
   - npm run dev

## Features
- React Router routing (Home, Search, Book Detail)
- Fetching data from an external API
- Loading and error handling states
- Simple styling/layout

## Challenges / Known Bugs
- Sometimes a book has no description, so the app shows "No description available."
